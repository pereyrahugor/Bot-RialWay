⚙️ Configuración del Sistema

Modifica los parámetros de configuración del bot. Estos cambios se aplicarán en Railway.

🤖 Configuración OpenAI

ASSISTANT_NAME

Nombre público que usará el asistente.

ASSISTANT_ID

ID del asistente principal en OpenAI.

ASSISTANT_ID_IMG

ID del asistente para procesamiento de imágenes.

OPENAI_API_KEY

Clave de API de OpenAI principal.

OPENAI_API_KEY_IMG

Clave de API de OpenAI para imágenes.

ID_GRUPO_RESUMEN

ID del grupo principal para reportes de ventas.

ASSISTANT_PROMPT

Configuración de la lógica maestra e instrucciones del sistema.

👤 Usuario Admin (Acceso CRM/Backoffice)

ADMIN_USER

Usuario de acceso al panel principal.

ADMIN_PASS

Contraseña de acceso al panel principal.

💬 Mensajes y Tiempos de Cierre

MSJCIERRE

Mensaje de despedida por inactividad.

TIMEOUTCIERRE

Tiempo (min) para cerrar por inactividad.

MSJSEGUIMIENTO1

Primer mensaje de seguimiento.

MSJSEGUIMIENTO2

Segundo mensaje de seguimiento.

TIMEOUTSEGUIMIENTO2

Tiempo (min) para el segundo seguimiento.

MSJSEGUIMIENTO3

Tercer mensaje de seguimiento.

TIMEOUTSEGUIMIENTO3

Tiempo (min) para el tercer seguimiento.

🌐 Servicios de Google

GOOGLE_CLIENT_EMAIL

Email de la cuenta de servicio.

GOOGLE_PRIVATE_KEY

Clave privada (formato JSON string).

GOOGLE_MAPS_API_KEY

API Key para Google Maps.

GOOGLE_CALENDAR_ID

ID del calendario de Google.

SHEET_ID_UPDATE

ID de la hoja de cálculo para actualizaciones.

SHEET_ID_RESUMEN

ID de la hoja de cálculo para resúmenes.

DOCX_ID_UPDATE

ID del documento para actualizaciones.

VECTOR_STORE_ID

ID del Vector Store de OpenAI.

⚡ Supabase

SUPABASE_URL

URL del proyecto Supabase.

SUPABASE_KEY

Service Role Key de Supabase.

🧭 Visibilidad de Navegación Hot-update

BACKOFFICE_VISIBLE

Mostrar o ocultar el acceso a Conversaciones en el Nav.

CRM_VISIBLE

Mostrar o ocultar el acceso al CRM en el Nav.

Editor de Lógica Maestra

Define el comportamiento, personalidad y reglas del bot con precisión quirúrgica.

Al publicar, los cambios se envían a la base de datos y a OpenAI automáticamente.

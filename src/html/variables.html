⚙️ Editar Variables de Entorno

Modifica los parámetros de configuración del bot. Estos cambios se aplicarán en Railway.

ASSISTANT_ID

ID del asistente principal en OpenAI.

OPENAI_API_KEY

Clave de API de OpenAI para el funcionamiento del bot.

ASSISTANT_NAME

Nombre público que usará el asistente.

BOT_NAME

Nombre interno para identificar la instancia del bot.

MSJCIERRE

Mensaje de despedida por inactividad.

MSJSEGUIMIENTO1

Primer mensaje de seguimiento automático.

TIMEOUTCIERRE

Tiempo en minutos para cerrar la sesión por inactividad.

ID_GRUPO_WS

ID del grupo principal de WhatsApp.

ID_GRUPO_RESUMEN

ID del grupo para reportes de ventas.

SHEET_ID_UPDATE

IDs de Google Sheets (separados por coma).

GOOGLE_CALENDAR_ID

ID del calendario de Google para agendamientos.

RAILWAY_TOKEN

Token de API de Railway.

RAILWAY_PROJECT_ID

ID del proyecto en Railway.
